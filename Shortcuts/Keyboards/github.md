### Site wide shortcuts
```
s or /     - Focus search bar
g + n      - Go to notifications
g + d      - Go to Dashboard
?          - Bring up the help dialog
j          - Move selection down
k          - Move selection up
x          - Toggle selection
o or enter - Open selection
h          - Open hovercard
```

### Repositories
```
Cmd + /    - Focus secondary search bar
g + c      - Go to code
g + i      - Go to issues
g + p      - Go to pull requests
g + b      - Go to projects
g + w      - Go to wiki
```

### Source code browsing
```
t         - Activates the file finder
l         - Jump to line
w         - Switch branch/tag
y         - Expand URL to its cononical form
i         - Show/hide all inline notes
b         - Open blame
```

### Mixin
```
Shift + /  - open the keyboard shortcut list
```
